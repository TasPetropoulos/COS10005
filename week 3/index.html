Campus Tour Robot

Introduction

This project is going to design and develop a life-size tour guide robot called Robby. Robby is capable of verbal and physical interaction with prospective students and visitors.

Team Members

The team members are all Swinburne students. Please click here to see details of the members.

Project Phases

This project has three phases:

  1. Design
  2. Implementation
  3. Testing
